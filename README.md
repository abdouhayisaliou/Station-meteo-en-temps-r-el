# 🌦️ Station Météo Connectée

Ce projet est une station météo connectée permettant de mesurer et afficher des données météorologiques en temps réel (température, humidité, pression, etc.). Les données peuvent être enregistrées pour suivre l’évolution dans le temps.

📌 Fonctionnalités

- Mesure de la température
- Mesure de l’humidité
- Mesure de la pression atmosphérique
- Affichage des données en temps réel
- Enregistrement des données (optionnel)
- Possibilité d’envoyer les données vers une base de données / serveur (optionnel)

🛠️ Matériel utilisé

- Microcontrôleur (Arduino / ESP32 / Raspberry Pi Pico…)
- Capteur DHT11 / DHT22 (température + humidité)
- Capteur BMP180 / BMP280 (pression + température)
- Écran LCD / OLED (optionnel)
- Breadboard + câbles

💻 Technologies / Langages

- C / C++ (Arduino IDE) ou MicroPython
- Serial Monitor / affichage LCD-OLED
- (Optionnel) MQTT / HTTP pour l’envoi de données
